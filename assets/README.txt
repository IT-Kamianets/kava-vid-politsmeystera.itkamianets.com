Помістіть сюди ваші фотографії для галереї.
Файли повинні називатися за бажанням, але у HTML-коді приклад:
assets/placeholder1.jpg, assets/placeholder2.jpg, тощо.
У кожному блоці галереї 3 фото. Вони автоматично будуть перемикатися за допомогою стрілок.

ЛОГОТИП
----------
Щоб додати логотип кав'ярні у навігацію, покладіть файл логотипу у цю папку з ім'ям:

assets/logo.png

Рекомендації:
- Формат: PNG (з прозорим фоном) або JPG
- Розмір: приблизно 200–500px по ширині (він буде відображатися з висотою ~42px в навігації)
- Ім'я файлу має бути `logo.png` або змініть шлях у `index.html` відповідно.

Після додавання `logo.png`, оновіть сторінку в браузері — логотип з'явиться у верхній навігації.
